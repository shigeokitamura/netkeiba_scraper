# Netkeiba Scraper
## Overview
netkeiba.com からデータをスクレイピングし、csvとして保存します。

## Description
保存される列は以下の通りです。

* 年
  * 1986-2020
* 月
  * 1-12
* 日
  * 1-31
* 開催場所
  * 札幌、函館、福島、新潟、東京、中山、中京、京都、阪神、小倉
* レース名
  * 3歳新馬, 4歳未勝利, 等
* 何レース目
  * 1-12
* コース
  * 芝、ダート
* 方向
  * 右回り、左回り、直線
* 距離（m）
  * 1000-4000
* 天候
* 馬場状態
* 着順
* 枠番
* 馬名
* 性別
* 年齢
* 斤量
* 騎手名
* タイム
* 上り
* オッズ（単勝）
* 人気
* 馬体重
* 馬体重の増減
* 調教師


## Requirements
```
python >= 3.7.3
pandas >= 0.24.2
beautifulsoup4 >= 4.7.1
```

## Usage
```
$ python scraping.py
```
